# EC2 – Servidors d’Aplicació WordPress

## Descripció

Amazon EC2 s’utilitza per crear les màquines virtuals on s’executa WordPress.  
Aquestes instàncies formen part d’un **Auto Scaling Group**, que permet augmentar o reduir el nombre de servidors segons la càrrega del sistema.

Aquest model permet implementar **escalabilitat horitzontal**, on es creen noves instàncies quan augmenta el tràfic de la web.

---

## Configuració utilitzada

- Sistema operatiu: Amazon Linux
- Tipus d’instància: t3.micro
- Accés SSH: port 22
- Accés web: port 80
- Security Group configurat per permetre tràfic HTTP , HTTPS i SSH

---

## Instal·lació de WordPress

A cada instància EC2 s’ha instal·lat:

- Apache / Nginx
- PHP
- Client MySQL
- WordPress

Aquesta instal·lació permet executar el servidor web i connectar-se amb la base de dades RDS.

## Connexió al meu EC2

per connectar-nos per ssh a la màquina virtual EC2, fem servir les següents ordres :
- Veiem que tenim la clau al nostre directori
- Donem permisos de lectura amb **chmod 400**
- Ens connectem amb *ssh*
