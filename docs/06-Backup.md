# AWS Backup

## Descripció

AWS Backup s’utilitza per crear còpies de seguretat automàtiques dels recursos del sistema.

Aquest sistema permet recuperar dades en cas de fallada o pèrdua d’informació.

---

## Recursos protegits

En aquest projecte es realitzen còpies de seguretat de:

- Base de dades RDS
- Sistema de fitxers EFS

---

## Configuració

S’ha configurat un pla de backup amb còpies periòdiques per garantir la protecció de les dades.

---

## Avantatges

- Protecció davant pèrdua de dades
- Recuperació davant desastres
- Gestió centralitzada de backups
